# ATTND Insight

ATTNDInsight is an Android attendance management application built using Java and Android Studio.

## Features

- Attendance data management
- Faculty-wise attendance tracking
- Overall attendance dashboard
- Download and export support
- WebView-based modern interface
- Local HTML dashboard integration
- File download support
- Responsive UI
- Android 8.0+ support
- Get Attendance data from ATTND app

## Tech Stack

- Java
- Android Studio
- WebView
- HTML
- CSS
- JavaScript

## Requirements

- Android 8.0 (API 26) or higher
- Internet permission
- Storage permission (for exports/downloads)
- ATTND (https://github.com/kachariyaparth2007-tech/Attendance-App.git)

## Installation

1. Clone the repository

```bash
git clone https://github.com/kachariyaparth2007-tech/Attendance-App.git
```

2. Open the project in Android Studio

3. Edit javascript and enter your firebase api ( instruction are written in script.js in " const firebaseConfig() " )
4. Sync Gradle and Run
